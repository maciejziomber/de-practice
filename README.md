\# Simple Python Data Pipeline

A basic ETL (Extract, Transform, Load) project.



\## How it Works

1\. \*\*Extract\*\*: Reads raw data from `data/raw\_data.csv`.

2\. \*\*Transform\*\*: Converts city names to UPPERCASE using Pandas.

3\. \*\*Load\*\*: Saves the result as a JSON file in `data/`.



\## Requirements

\- Python 3.x

\- Pandas (`pip install pandas`)
